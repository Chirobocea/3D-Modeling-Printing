
# Mobile Phone Tripod
This is the final project for 3D Modeling and Printing course from University of Bucharest.
The project consist of 4 checkpoints and final project. 

# Description
This project comes with 2 versions of mobile phone tripod:
* __Tilt & Roll__ version
* __Ball__ version

This versions are made to be chosen by:
* Photography & videography preferences
* The need for versatility & fast use
* The need of rigidity & precision

# Mobile Tripod Tilt & Roll
* __Stabel__
* __Versatile__
* __Precies__
* 100% __3D printable__

The main 3 axis:
* __Pan__  - may be controlled by rotating the tripod
* __Tilt__ - direct control by screw
* __Roll__ - direct control by screw

Phone tolerance:
* __Minimum__ phone width: 52mm
* __Maximum__ phone width: 77mm

Tripod hight:
* Adjustable in 3 steps
* Fixed angle leg: 60 degree
* __Minimum__: 235mm
* __Maximum__: 280mm
