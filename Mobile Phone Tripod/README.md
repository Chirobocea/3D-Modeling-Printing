
# Mobile Phone Tripod
<img src="https://user-images.githubusercontent.com/72545287/119258336-e3d6db80-bbd1-11eb-8d74-8a8f9f71de7c.png" width="300"/>

This is the final project for 3D Modeling and Printing course from University of Bucharest.

The project consist of 3 checkpoints and final project:
* __Checkpoint 1__: make components (empty) of tripod
* __Checkpoint 2__: make skatches, body, joints, motion studies; add apperance and rander
* __Checkpoint 3__: finish the second version: ball version

I chose this project because I am passionate about photo-video equipment and I wanted to made my own

# Description
This project comes with 2 versions of mobile phone tripod:
* __Axes__ version
* __Ball__ version

This versions are made to be chosen by:
* Photography & videography preferences
* The need for versatility & fast use
* The need of rigidity & precision

# Mobile Tripod Axes
* __Stabel__
* __Versatile__
* __Precies__
* __100% 3D printable__

The main 3 axis:
* __Pan__  - may be controlled by rotating the tripod
* __Tilt__ - direct control by screw
* __Roll__ - direct control by screw

Phone tolerance:
* __Minimum__ phone width: 52mm
* __Maximum__ phone width: 77mm

Tripod hight:
* Adjustable in 3 steps
* Fixed angle leg: 60 degree
* __Minimum__: 235mm
* __Maximum__: 280mm

# Assembly recomandation
* Use two springs for holder phone to make it easier to open the case
* For ball version use a 3mm radius, 12-15mm heigh screw for phone holder

# Files
* __F3D__ for both versions
* __STL__ by components for both versions - the "x3" in the name of a stl means that it needs to be printed 3 times
